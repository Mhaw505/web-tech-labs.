Exercise 7 - Font Properties

1. Font Family

Playfair Display — Google Font, elegant serif style

Lato — Google Font, clean modern sans-serif

Source Code Pro — Google Font, monospace for code

Georgia — web safe serif font

Arial — web safe sans-serif font

2. Font Size

font-size: 24px — fixed pixels

font-size: 1.5em — relative to parent

font-size: 1.5rem — relative to root

font-size: 130% — percentage

font-size: 0.8rem — smaller than normal

3. Font Weight

font-weight: 100 — Thin

font-weight: 300 — Light

font-weight: 400 — Normal

font-weight: 700 — Bold

font-weight: 900 — Black

4. Font Style and Variant

font-style: normal — regular text

font-style: italic — slanted text

font-style: oblique — also slanted

font-variant: small-caps — This Makes Lowercase Look Like Small Capitals
