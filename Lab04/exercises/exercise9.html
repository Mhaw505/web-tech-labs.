Exercise 9 - Text Spacing and Layout

1. Line Height

line-height: 1 (tight)

This paragraph has a very tight line height of 1. The lines are cramped together and hard to read when there is a lot of text like this.


line-height: 1.6 (comfortable)

This paragraph has a comfortable line height of 1.6. This is the recommended value for body text because it is easy to read.


line-height: 2.5 (spacious)

This paragraph has a very large line height of 2.5. The lines are very far apart and feel very airy and open.

2. Letter Spacing

letter-spacing: -2px — letters are squeezed together

letter-spacing: 0 — default normal spacing

letter-spacing: 4px — letters spread apart

letter-spacing: 8px — very wide spacing

3. Word Spacing

word-spacing: -4px — words are closer together

word-spacing: 0 — default normal spacing

word-spacing: 15px — words are far apart

4. Text Indent

text-indent: 60px — this paragraph has a much larger indent of 60 pixels on the first line only.
