Main Content

This holy grail layout is built using CSS floats. The nav and aside are floated left with fixed percentages, and the main content takes the remaining space.

Floats were the original way to build multi-column layouts before Flexbox and Grid were introduced. They require clearfix hacks to prevent collapse.

Main Content

This holy grail layout is built using Flexbox. The three columns are flex items inside a flex container. The main content uses flex: 1 to fill remaining space automatically.

Flexbox makes equal height columns effortless — something floats struggled with. It also handles reordering columns easily with the order property.

Main Content

This holy grail layout is built using CSS Grid. The entire layout is defined with grid-template-areas giving each section a named area — header, nav, main, aside, footer.

CSS Grid is the most powerful method. It controls both rows and columns simultaneously, making complex layouts simple and readable with just a few lines of CSS.
